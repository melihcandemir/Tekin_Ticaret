# Tekin Ticaret - Veresiye Defteri

Tekin Ticaret, müşterilerle aranızdaki güven bağını dijitale taşıyan modern bir **Veresiye Defteri** uygulamasıdır. Müşteri borçlarınızı kolayca takip etmenizi sağlar.

## 🚀 Özellikler

- **Müşteri ve Borç Yönetimi:** Müşterilerinizi sisteme ekleyin, güncel borç ve ödemelerini kolayca takip edin.
- **Güvenli Kimlik Doğrulama:** Supabase altyapısı ile yetkisiz erişimlere karşı güvenli giriş sistemi.
- **Modern ve Şık Arayüz:** Tailwind CSS ile geliştirilmiş, her cihaza tam uyumlu (responsive) kullanıcı dostu tasarım.
- **İletişim & Harita:** Kolay erişilebilir iletişim bilgileri ve entegre Google Haritalar yönlendirmesi.

## 🛠 Kullanılan Teknolojiler

- **Frontend:** React (Vite), Tailwind CSS, React Router DOM
- **Backend & Veritabanı:** Supabase
